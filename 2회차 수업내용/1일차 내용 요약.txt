변수
 - 원하는 데이터를 관리할 수 있는 변수를 선언할 수 있다
 - 변수 선언 방법
자료형 변수명 = 초기값;
자료형 변수명;
 - 자료형의 종류 : char, int, float, double
 - 변수명 작성 규칙/관례
 - 변수명 입출력 : printf, scanf
   - 서식문자 종류

연산자
- 연산자 종류를 알고, 결과를 예측할 수 있다.
- 산술연산자 : + - * / %
  - / : 정수 / 정수 => 정수
  - 자료형변환이 필요
  - % : 나머지 연산자
- 대입연산자 :  =
- 증감연산자 : ++, -
  - 전위형, 후위형
  - 최종적으로 1증가/감소
- 비교연산자 : > < >= <= == !=
  - 결과가 참(0이 아닌수) 또는 거짓(0)
